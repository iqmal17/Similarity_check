Kenapa SimCheck?

SimCheck dibuat untuk dosen & panitia akademik yang ingin cek kemiripan tugas mahasiswa secara cepat, adil, dan transparan. Algoritma gabungan Jaccard + Cosine TF-IDF + N-gram memberi skor yang mudah dibaca.

  • ✓ Skor % jelas: Rendah / Sedang / Mirip Tinggi
  • ✓ Sorotan kalimat mirip side-by-side
  • ✓ Heatmap Matrix seluruh kelas
  • ✓ Export Detail: Excel, PDF, Word — siap arsip
  • ✓ 3 mode sensitivitas
  • ✓ 100% lokal — privasi tugas terjaga
